① 直接集成

```js
// 安装依赖
// npm install @mykurisu/calendar-component-miniapp

// index.json
{
  "usingComponents": {
    "calendar": "@mykurisu/calendar-component-miniapp"
  }
}

// index.wxml
<view>
  <calendar needHeader />
</view>
```
